# Security Policy

## Supported Scope

`slides2pdf` is a browser-based static tool. Videos are intended to be processed locally in the user's browser without uploading them to a server.

## Reporting a Vulnerability

If you find a security issue, please open a GitHub issue with enough detail to reproduce the problem. If the issue involves private data exposure, avoid posting sensitive files or private videos publicly.

Useful details include:

- Browser and operating system
- Steps to reproduce
- Whether a file is uploaded or transmitted unexpectedly
- Any console errors

## Privacy Expectations

The project should not introduce server-side upload, telemetry, or third-party transfer of video files without clear documentation and explicit user consent.

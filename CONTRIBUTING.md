# Contributing

Thanks for your interest in improving `slides2pdf`.

This project is a small browser-based tool for extracting slide-like frames from videos. Contributions that improve reliability, usability, privacy, and research workflows are welcome.

## Good First Contributions

- Improve README examples
- Add screenshots or short demo videos
- Improve UI text
- Test with different kinds of lecture or webinar videos
- Report cases where slides are missed or too many frames are captured
- Improve accessibility labels and keyboard navigation

## Development Notes

The current implementation is a static HTML file using browser APIs. It does not require a backend.

Before proposing a change, please check:

- The tool still runs without installation
- Videos are processed locally in the browser
- No external upload is introduced
- Output still works for image ZIP and PDF generation
- The UI remains understandable for non-technical users

## Issue Reports

When reporting a problem, include:

- Browser and operating system
- Video length and approximate resolution
- What you expected
- What happened instead
- Whether sensitivity settings changed the result

Do not upload private or copyrighted videos unless you have permission to share them.

## Pull Requests

Please keep pull requests focused. For larger changes, open an issue first and describe the proposed workflow.
